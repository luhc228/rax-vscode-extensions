# rax-vscode-extensions

Rax VS Code Extensions

## 1. vscode-rax-snippets
### Snippets

| Snippet | Renders                     |
| ------- | --------------------------- |
| `ir`    | Import Rax                  |
| `irc`   | Import Rax / Component      |
| `irh`   | Import Rax / Hooks          |
| `cc`    | Class Component             |
| `fc`    | Function Component          |
| `ush`   | Use State Hook              |
| `ueh`   | Use Effect Hook             |
| `umh`   | Use Memo Hook               |
